Ayuda para el Formulario de Configuración

A continuación se explicarón los apartados que componen el formulario de configuración de la aplicación Interfaz para Planta Piloto.

A este formulario se puede llegar de dos formas:

Formulario de Configuración a través de la opción de menú "Crear configuración"

En esta ventana se creará una nueva configuración para un proyecto.

  1. Apartado que contiene los datos generales del proyecto (nombre, descripción...).
  2. Apartado que contiene las propiedades necesarias para la definión de una nueva variable.
  3. Botón que evalúa y guarda la variable que se está creando y se asigna al proyecto que se está creando. Tras esto, los valores se limpian y se quedan preparados para guardar una nueva variable. Los valores de una variable no se guardarán si no se pulsa este botón.
  4. Botón de acceso de ayuda de este formulario.
  5. Botón que guarda la configuración creada en el archivo de texto que se especifique en el cuadro de diálogo que se abre al pulsar el botón, y carga dicho proyecto en la aplicación.
  6. Botón que cancela la creación del proyecto, perdiéndose todos los datos definidos y volviendo a la ventana principal.

Formulario de Configuración a través de la opción de menú "Modificar configuración"

En esta ventana podremos modificar la configuración del proyecto ya cargado en la ventana principal. Esto significa que podremos modificar las propiedas del proyecto y de las variables que tiene asociadas, pero no se podrá agregar nuevas variables.

  1. Apartado que contiene los datos generales del proyecto (nombre, descripción...).
  2. Apartado que contiene las propiedades necesarias para la definión de una nueva variable.
  3. Menú desplegable que contiene las variables definidas en el proyecto. A través de este menú elegiremos la variable que queremos modificar (los datos se cargarán automáticamente). Los nuevos valores que se asignen a las propiedades de la variable en edición se guardará al elegir otro elemento en el menú desplegable o al pinchar en el botón "Aceptar".
  4. Botón de acceso de ayuda de este formulario.
  5. Botón que guarda la configuración creada en el archivo de texto que se especifique en el cuadro de diálogo que se abre al pulsar el botón, y carga dicho proyecto en la aplicación.
  6. Botón que cancela la creación del proyecto, perdiéndose todos los datos definidos y volviendo a la ventana principal.
