Ayuda para el Formulario Selección de Variables

A continuación se explicarán los apartados que componen el formulario de selección de variables de la aplicación Interfaz para Planta Piloto. Esta ventana puede presentarse de tres formas diferentes que se explicarán a continuación.

Formulario Selección de Variables a través del botón "Variables"

En esta ventana se seleccionarán las variables cargadas en el proyecto de las que se quieran visualizar los valores.

  1. Columna con los nombres de todas las variables cargadas en la configuración de la aplicación.
  2. Columna en la que se seleccionarán las variables que se quieren mostrar..
  3. botón de ayuda de la ventana.
  4. botón para mostrar las variables seleccionadas.
  5. botón para cerrar el formulario

Formulario Selección de Variables a través del botón "Gráfico"

En esta ventana se seleccionarán las variables que quieren ser graficadas en tiempo real. Crearemos tantas gráficas como desemos siempre que pulsemos el botón "Crear Gráficas", pudiendo cambiar la selección de variables entre ellas.

  1. Columna con los nombres de todas las variables cargadas en la configuración de la aplicación que sean graficables (que no son cadenas de texto).
  2. Columna en la que se seleccionarán las variables que se quieran graficar.
  3. botón de ayuda de la ventana.
  4. botón para crear la gráfica con las variables seleccionadas.
  5. botón para cerrar el formulario

Formulario Selección de Variables a través del botón "Archivo"

En esta ventana se seleccionarán todas las variables que se quieran guardar en un archivo de texto.

  1. Columna con los nombres de todas las variables cargadas en la configuración de la aplicación.
  2. Columna en la que se seleccionarán las variables que se quieran guardar en un archivo.
  3. botón de ayuda de la ventana.
  4. botón para comenzar el guardado de las variables seleccionadas.
  5. botón para cerrar el formulario

En el momento que pulsemos el botón "Crear archivo" con alguna de las variables seleccionadas se nos abrirá el siguiente cuadro de diálogo donde elegiremos la ruta en la que guardar el archivo de texto que contiene los valores de las variables que hemos seleccionado.

Por defecto, el nombre que se le asignará al archivo será "nombre_del_proyecto_nombre_de_cada_variable_seleccionada.txt".

Formato del archivo donde se guardarán las variables

Al abrir el archivo que acabamos de crear y en donde se guardarán los valores de las variables seleccionadas nos encontraremos:

  1. Fecha de creación del archivo.
  2. Nombre del proyecto.
  3. Descripción del proyecto.
  4. Ruta de la imagen cargada en el proyecto.
  5. Lista de variables que se están guardando, separadas por ';'.
  6. Serie de valores devueltos por la placa de las variables seleccionadas, separados por ';'.
