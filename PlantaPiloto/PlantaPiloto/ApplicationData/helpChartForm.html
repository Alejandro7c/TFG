Ayuda para el Formulario Gráficas

A continuación se explicarán los apartados que componen el formulario "Gráficos" de la aplicación Interfaz para Planta Piloto.

Ventana "Gráfico"

  1. Eje Y: muestra los valores que alcanzan las variables graficadas.
  2. Eje X: muestra el momento en segundos desde que la placa ha sido encendida.
  3. Lista de variables graficadas.
  4. Cantidad de valores que se quieren mostrar de las variables.
  5. Botón de acceso a la ayuda de este formulario.
  6. Botón para cerrar la ventana.
