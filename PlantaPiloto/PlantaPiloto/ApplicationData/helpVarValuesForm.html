Ayuda para la ventana Valores de variables

A continuación se explicarán los apartados que componen la ventana "Valores de vaiables" de la aplicación Interfaz para Planta Piloto.

Ventana "Valores de variables"

  1. Columna con las variables seleccionadas.
  2. Columna con los valores actualizados de cada variable seleccionada. Estos valores se actualizarán cada 5 segundos.
  3. Botón de acceso a la ayuda
  4. Botón para cerrar la ventana.
